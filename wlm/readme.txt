2014/9/17 a.ide

ssh転送

以下の環境の場合。
・中継サーバ: 10.124.94.134
・ターゲット: 172.20.53.12

windows
1. teratermでssh転送の設定をする。
2. 中継サーバへログインする。

linux
1. linuxサーバから中継サーバへ以下のコマンドを実行する。
  ssh -N -f -L 15000:172.20.53.12:5000 10.124.94.134
  ssh -N -f -L 18000:172.20.53.12:8004 10.124.94.134


2. webサーバの起動/停止/再起動
  /etc/init.d/httpd [start | stop | restart]
